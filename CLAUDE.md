# Trace Type プロジェクト用 Claude 指示書

## プロジェクト概要
日本語タイピング練習用のWebアプリケーションです。ビジュアルフィードバックとキーガイダンス機能を通じて、ユーザーのタイピングスキル向上をサポートします。

## プロジェクト構成
- **index.html**: タイピングエリアとコントロールを含むメインHTMLインターフェース
- **script.js**: タイピング練習のコアJavaScriptアプリケーションロジック
- **styles.css**: タイピングインターフェースのCSSスタイリング
- **Practice/**: 練習テキストファイル（.md形式）を格納するディレクトリ
- **Practice/files.json**: 利用可能な練習ファイルをリストするJSONファイル

## 主要機能
- **ビジュアルタイピングフィードバック**: 正確性に基づいて文字の色が変化（緑=正解、赤=不正解、灰色=未入力）
- **キーガイダンス**: JISキーボードマッピングを使用して各文字に対応するキーを表示
- **ロック機能**: 正しく入力された文字は「ロック」され、削除できない
- **マウスクリック追跡**: 練習中のマウスクリック回数をカウント・表示
- **IMEサポート**: 日本語入力メソッドの変換イベントを処理
- **タイマー**: 練習セッション中の経過時間を追跡

## 技術的詳細
- Fetch APIを使用して練習ファイルとファイルリストを読み込み
- ビジュアルフィードバック付きの文字単位バリデーションを実装
- ローマ字と日本語文字入力の両方をサポート
- 純粋なタイピング練習を確保するためコピー/ペーストとその他のショートカットを防止
- 現在の文字に対するリアルタイムキーガイダンス表示

## 開発ガイドライン
- インターフェース全体で日本語サポートを維持
- 正しく入力された文字のロック機能を保持
- ビジュアルフィードバックシステムの一貫性を保つ
- アクセシビリティとキーボードのみのナビゲーションを確保
- 様々な日本語入力メソッド（IME）でテスト

## テスト項目
- Practiceディレクトリの異なる練習ファイルでテスト
- マウスクリックカウントが正しく動作することを確認
- IME変換処理をテスト
- タイピング中にビジュアルフィードバックが適切に更新されることを確認
